Pasos para conectar
1. Accede a TikTok Ads Manager
2. Ve a Assets → Aplicaciones y crea una App
3. Genera un Access Token de larga duración
4. Copia tu Advertiser ID
5. Díselo a Claude Code → genera el Apps Script listo
